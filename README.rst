Install
------------------------------------------------------------------------------

``learn_unstructured`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-unstructured

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-unstructured
